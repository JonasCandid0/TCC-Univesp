# Análise Espacial e Temporal de Encalhes de Fauna Marinha no Litoral de São Paulo

Este repositório contém o código e a estrutura de dados do meu Trabalho de Conclusão de Curso (TCC). O projeto utiliza dados do **Sistema SIMBA** para analisar padrões de encalhes de fauna marinha, aplicando técnicas de ciência de dados e geoprocessamento.

## Objetivo do Projeto
Realizar uma análise detalhada sobre a ocorrência de encalhes de fauna no litoral paulista, identificando:
* Hotspots geográficos de encalhes.
* Tendências temporais (sazonalidade e variações anuais).
* Correlações entre o esforço de monitoramento e os registros encontrados.

## Estrutura do Repositório
* `data/`: Contém os dados brutos (originais) e processados.
* `notebooks/`: Jupyter Notebooks com as análises passo a passo.
* `requirements.txt`: Lista de bibliotecas Python necessárias para rodar o projeto.

## Ferramentas Utilizadas
* **Linguagem:** Python 3.12
* **Bibliotecas Principais:** * `Pandas` (Manipulação de dados)
    * `GeoPandas` (Análise espacial/GIS)
    * `Matplotlib` e `Seaborn` (Visualização de dados)
    * `NumPy` (Cálculos matemáticos)
